# C++ wav file player with rythm 

The project asks you to press enter to a rythm, press q when you are done, then program plays back a bell wav file to a rythm you entered

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download C++ compiler
2.Open program
3. Run the main.cpp

Instructions for using the software:

1. Press enter to start recording
2. input enter to a rythm
3. enter q to quit

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* mmsystem
* random
* chrono
